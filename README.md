# Quantum Device Design
<img width="1208" height="870" alt="image" src="https://github.com/user-attachments/assets/05e8e95e-dc9a-4a2b-8a11-d9405fee0837" />

This project models a single transmon qubit coupled to a CPW meander resonator using Qiskit Metal and scqubits. 
Starting from physical layout design, it performs a parametric sweep of the transmon pad height to 
characterize how qubit frequency and anharmonicity vary with geometry.
This demonstrates the full physical-layout → quantum Hamiltonian analysis pipeline 
without requiring EM simulation software (Ansys).

Check out the jupyternotebook file in this repository to read in detail
